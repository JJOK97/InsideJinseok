주요 기능 소개

PICPLE은 실시간 화상 연결을 통해 함께 사진을 찍고, AI 기술로 다양한 배경을 생성하며, 추억을 저장하고 공유할 수 있는 기능을 제공합니다.

실시간 크로마키 포토부스
  • 실시간 웹캠 화면 크로마키 처리
  • 인물 크기 조절 및 위치 이동
  • 다중 사용자 동시 촬영
  • 실시간 프리뷰 제공
AI 배경 생성 및 커스텀
  • AI 기술을 활용한 배경 이미지 생성
  • 텍스트 프롬프트 기반 이미지 생성
  • 로컬 이미지 파일 업로드
  • 배경 이미지 실시간 적용
추억 저장 및 공유
  • 개인 캘린더 자동 저장
  • 날짜별 추억 관리
  • 커뮤니티 게시판 공유
  • 소셜 미디어 공유 기능
실시간 화상 연결
  • WebRTC 기반 실시간 통신
  • 다자간 화상 연결 지원
  • 실시간 채팅 기능
  • 네트워크 상태 모니터링
